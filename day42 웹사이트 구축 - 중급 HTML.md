##### 머릿글 `<thead>` 안에 넣기 VS main table 안에 넣기
보이는 것은 동일하나 스타일 지정을 다르게 한다거나 기능을 다르게 하는 데에 있어 구분이 필요하기 때문에 `<thead>`안에 넣음

```
<thead>
	<tr>
	  <th>Dates</th>
	  <th>Work</th>
	</tr>
</thead>
```
